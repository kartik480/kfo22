Test RBH My SDSA Employee Users API

This API fetches SDSA employee users from tbl_sdsa_emp_users who are reporting to the currently logged-in user.

What This API Does

Updated Logic for SDSA Employee Users:

  • Input: User ID or username from login
  • Process: Finds logged-in user in tbl_user
  • Search: Looks in tbl_sdsa_emp_users for reportingTo = user_id
  • Result: Returns SDSA employee users reporting to that user ID

Key Difference: This API uses tbl_sdsa_emp_users table (SDSA employees) instead of tbl_sdsa_users
