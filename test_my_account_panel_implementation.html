Navigation & Data Flow

Navigation Flow:

  1. User clicks account icon in Business Head panel
  2. MyAccountPanelActivity opens with user data passed via Intent
  3. User can navigate through different account sections
  4. Back button returns to Business Head panel
  5. Logout redirects to login screen

Data Passing:

  • USER_ID: User's unique identifier
  • USERNAME: User's username
  • FIRST_NAME: User's first name
  • LAST_NAME: User's last name
