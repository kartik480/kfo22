Endpoint: https://emp.kfinone.com/mobile/api/get_rbh_active_users.php
Method: GET
Parameters: URL Query Parameters
Purpose: Fetch active users reporting to a specific RBH user

üîß Postman Setup

Step 1: Open Postman and create a new request
Step 2: Set the request method to GET
Step 3: Enter the URL with parameters:
https://emp.kfinone.com/mobile/api/get_rbh_active_users.php?reportingTo=rbh_username&status=active
Step 4: No additional headers or body required for GET request

üìù Request Parameters

Parameter Type Required Description Example
reportingTo string Yes Username of the RBH user whose subordinates to fetch "john_doe"
status string No User status filter (default: "active") "active"

üîç Database Query

The API executes this SQL query against the tbl_rbh_users table:
SELECT first_name, last_name, username, Phone_number, email_id, status FROM tbl_rbh_users WHERE reportingTo = :reportingTo AND status = :status ORDER BY first_name, last_name
