API Endpoint: get_rbh_manage_icons.php

This API fetches manage icons for RBH users based on their manage_icons column in tbl_user.

üìã API Documentation

Endpoint:

POST /api/get_rbh_manage_icons.php

Request Body:

{
    "user_id": "40",
    "username": "test_user"
}

Response Format:

{
    "status": "success",
    "message": "Manage icons fetched successfully",
    "data": [
        {
            "id": "1",
            "icon_name": "Dashboard",
            "icon_url": "https://example.com/dashboard",
            "icon_image": "dashboard.png",
            "icon_description": "Access main dashboard",
            "status": "Active"
        }
    ],
    "debug_info": { ... }
}

Database Tables Used:

  • tbl_user - Contains manage_icons column with comma-separated icon IDs
  • tbl_manage_icon - Contains icon details (name, URL, image, description, status)

How It Works:

  1. API receives user_id and username
  2. Fetches manage_icons from tbl_user for the given user
  3. Parses comma-separated icon IDs
  4. Fetches icon details from tbl_manage_icon using those IDs
  5. Returns only active icons with full details
