Managing Director APIs Test

1. Test ManagingDirectorUsers API (Dropdown Users)

This API fetches designated users for the Managing Director dropdown.

2. Test ManagingDirector API (Team Members)

This API fetches users who report to a selected designated user.

3. Database Structure Information

Tables Used:

  • tbl_user - Contains user information with columns: id, username, firstName, lastName, mobile, email_id, reportingTo, designation_id, status
  • tbl_designation - Contains designation information with columns: id, designation_name

API Endpoints:

  • ManagingDirectorUsers.php - Fetches designated users for dropdown (CBO, RBH, Director, Managing Director)
  • ManagingDirector.php - Fetches team members who report to selected manager

Workflow:

  1. User selects a designated user from dropdown (loaded from ManagingDirectorUsers.php)
  2. User clicks "Show Data" button
  3. System fetches users who have reportingTo = selected user's ID (via ManagingDirector.php)
  4. Results are displayed in a list showing username, firstName, lastName, mobile, email_id
