Test Partner Users by CBO/RBH Creators API

API Description

This API fetches all partner users from tbl_partner_users who were created by users with designation "Chief Business Officer" or "Regional Business Head".

The API joins tbl_partner_users with tbl_user and tbl_designation to find creators' designations.

Test API

Click the button below to test the API and fetch partner users created by CBO and RBH users.

Postman URL

GET https://emp.kfinone.com/mobile/api/get_partner_users_by_cbo_rbh_creators.php

Expected Response Format

Response Structure:
- status: "success" or "error"
- message: Description of the operation
- data: Grouped by creator designation and username
- summary: Total counts and creator designations found
