Test RBH My Agent Data API

This test will fetch agents created by a Regional Business Head user from the tbl_agent_data table.

Key Features:

  • ✅ Fetches agents where createdBy matches the RBH username
  • ✅ Supports both User ID and Username parameters
  • ✅ Handles full names and numeric usernames
  • ✅ Provides comprehensive debug information
  • ✅ Returns agent statistics (total, active, inactive)
  • ✅ Flexible user detection for testing

API Information

Endpoint: https://emp.kfinone.com/mobile/api/rbh_my_agent_data.php

Method: GET

Parameters:

  • user_id - User ID of the RBH user (e.g., 23)
  • username - Username of the RBH user (e.g., 30000)

Android App Flow

Login Flow:

  1. User logs in with username (e.g., 30000)
  2. Login response: {"user":{"id":23,"username":"30000","firstName":"PREM SWAROOP REDDY","lastName":"YERRA","designation_name":"Regional Business Head"}}
  3. EnhancedLoginActivity passes username: "30000" to RegionalBusinessHeadPanelActivity
  4. RegionalBusinessHeadPanelActivity passes USERNAME: "30000" and USER_ID: "23" to RBHMyAgentPanelActivity
  5. RBHMyAgentPanelActivity passes USERNAME: "30000" and USER_ID: "23" to RBHMyAgentListActivity
  6. RBHMyAgentListActivity calls API with user_id=23
  7. API returns agents where createdBy = "30000"
