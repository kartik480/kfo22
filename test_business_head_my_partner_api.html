Method: GET

Purpose: Fetch all partner users created by a Business Head user from tbl_partner_users table

Table: tbl_partner_users (not tbl_agent_data)

Key Column: createdBy (contains usernames)

Database Schema - tbl_partner_users

Key Columns:

  • id - Primary key
  • username - Partner username
  • first_name, last_name - Partner full name
  • Phone_number - Contact number
  • email_id - Email address
  • company_name - Company name
  • status - Active/Inactive status
  • createdBy - Username of creator (Business Head)
  • created_at - Creation timestamp

Filtering Logic: WHERE createdBy = 'Business Head Username'
