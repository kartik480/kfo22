Test Partner Users by CreatedBy API

1. Get All Partner Users Grouped by CreatedBy

This will fetch all partner users grouped by their createdBy value, showing count and list of users for each creator.

2. Get Partner Users by Specific CreatedBy

Enter a createdBy value to fetch partner users created by that specific user.

3. Postman URLs

Get All Grouped by CreatedBy:

GET https://emp.kfinone.com/mobile/api/get_partner_users_by_created_by.php

Get by Specific CreatedBy:

GET https://emp.kfinone.com/mobile/api/get_partner_users_by_created_by.php?createdBy=11

Replace "11" with any createdBy value you want to test
